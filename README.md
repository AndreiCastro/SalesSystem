<h1>Sales System - MVP</h1>

### Status: Finalizado

### Projeto de um sistema de vendas com CRUD para: Cliente - Produto - Venda

### Cliente Model
* Id
* Nome
* E-mail
* CPF/CNPJ
* Logradouro
* Bairro
* UF
* Cep
* Cidade
* Telefone

### Produto Model
* Id
* Nome
* Descrição
* Preço
* Unidade de Medida
* Quantidade
* Peso
* Data de Validade

### Venda Model
* Id
* Data Venda
* Quantidade Produto Vendido
* Valor Total
* Descrição
* Desconto
